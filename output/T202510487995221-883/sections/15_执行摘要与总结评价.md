## 1. 执行摘要（Executive Summary）

### 项目定位与目标

本项目是一个**教学/实验性质的 RISC-V 操作系统内核**，目标架构为 `riscv64gc-unknown-none-elf`（64 位 RISC-V 裸机环境）。从 Git 历史记录（150+ 提交，8 周开发周期）推断，项目原计划实现完整的多架构（RISC-V + LoongArch）操作系统，支持 LTP 测试套件、lwext4 文件系统、VirtIO 驱动等核心功能。

### 技术栈概览

| 类别 | 详情 |
|------|------|
| **编程语言** | Rust (no_std) |
| **目标架构** | RISC-V 64 位 (riscv64gc-unknown-none-elf) |
| **构建工具** | Cargo（推断，Cargo.toml 缺失） |
| **外部依赖** | lwext4 文件系统（预编译静态库，未集成） |
| **开发工具链** | RISC-V GNU Toolchain（vendor 目录含预编译二进制） |
