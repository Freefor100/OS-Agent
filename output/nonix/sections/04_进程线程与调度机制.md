```rust
// os/src/fs/pipe.rs:309 (读管道阻塞时)
if buffer_is_empty {
    suspend_current_and_run_next();  // 阻塞当前任务，调度下一个
}

// os/src/fs/pipe.rs:340 (写管道阻塞时)

    suspend_current_and_run_next();  // 阻塞当前任务，调度下一个
}
```

**推断的状态流转**：
```
Running --[I/O 阻塞]--> Blocked --[I/O 完成]--> Ready --[调度]--> Running
```
